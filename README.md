# QuingKhaos' Factorio Library

A set of commonly-used utilities by QuingKhaos for creating Factorio mods.

## Stability Guarantee

khaoslib follows [Semantic Versioning 2.0.0](https://semver.org/spec/v2.0.0.html) with specific considerations for the
Factorio modding ecosystem.
